# @ManyToOne(FetchType)

## 🎊알아두면 좋은 지식

**프록시(가짜) 객체 특징**

프록시 객체는 처음 사용할 때 한 번만 초기화 된다.

프록시 객체를 초기화 할 때, 실제로 엔티티로 바뀌는 것이 아니다.

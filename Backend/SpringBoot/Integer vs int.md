# Integer vs int

## 📌 정의

### int 

+ **primitive 자료형**이다. (long, float...)
+ 산술 연산이 가능하다.
+ **null로 초기화 할 수 없다. **

### Integer

+ Unboxing을 하지 않으면 산술 연산이 불가능하다.
+ **null 값을 처리**할 수 있다. 
  + SQL과 연동할 경우 처리가 용이하다.

## 🎊 참고

### Boxing 

+ Primitive 자료형 → Wrapper 클래스

### Unboxing 

+ Wrapper 클래스 → Primitive 자료형

# 빌더 (Builder)

## 📌 정의

+ 객체를 직접 생성하는 대신 그 전에 **필수 인자들을 전달해 빌더 객체로 만들고** 메소드를 호출해 인스턴스를 생성한다.

**👍 장점**

+ 필요한 데이터만 사용할 수 있다. 

+ 새로운 변수가 추가되도 기존의 코드에 영향을 주지 않을 수 있다.

+ 의미하는 것을 파악하기 쉽다. (가독성을 높인다.)

  ```
  User user = User.builder()
               .name("son")
               .height(180)
               .build();
  ```

# 보안, 자격 증명 및 규정 준수

## Cloud Directory

## IAM(Identity And Acess Management)

AWS 서비스, 리소스에 대한 액세스를 안전하게 관리할 수 있다.

AWS 사용자, 그룹을 관리 생성/관리 할 수 있다.

권한에 따라 AWS 리소스 액세스 ALLOW/DENY 설정할 수 있다.

## Inspector

조사관, 감독관을 의미한다.

AWS에 배포 된 애플리케이션의 보안이나 규정 준수를 개선하는데 도움을 준다.

자동화된 보안평가 서비스이다.

# 표준스트림(standard streams)

## 📌정의

표준 스트림(standard streams)은 **특정한 프로그래밍 언어 인터페이스뿐만 아니라** 유닉스 및 유닉스 계열 운영 체제에서 **컴퓨터 프로그램과 그 환경 사이에 미리 연결된 입출력 통로**를 가리킨다.

## 📑사용

```
echo foo
```

`echo`는 다음 나오는 문자열을 터미널에 출력하는 명령어이다.
